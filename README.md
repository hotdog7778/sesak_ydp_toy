## textLED
- 설명 : 입력받은 문구에 사이즈,폰트,컬러를 설정해서 횡이동 하는 애니메이션을 화면에 보여주는 웹 페이지
- 목적 : HTML, CSS, Js, Bootstrap, Jquery를 사용해서 간단한 반응형 페이지를 만들기.

추가할 것
